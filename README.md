# IIT_Hyderabad_Internship-26

## 📌 Overview

This repository contains the work completed during my internship at the Indian Institute of Technology Hyderabad (IITH), focusing on advanced research and practical implementation in photonics, semiconductor fabrication, and sensor system design.

## 🎯 Internship Objective

The objective of this internship was to gain hands-on experience in cutting-edge technologies involving optical waveguides and microfluidic sensing systems, along with understanding real-world fabrication and design methodologies.

## 🛠️ Domains Covered

* Silicon Photonics
* Semiconductor Fabrication
* Microfluidics
* Sensor Design
* Integrated Systems

## 📂 Repository Structure

### 🔹 Project 1: SOI Strip Optical Waveguide Fabrication

* **Description:** Design and fabrication of SOI-based strip optical waveguides for light propagation in photonic circuits.
* **Focus Areas:** Photolithography, etching processes, and waveguide design
* **Applications:** Optical communication, integrated photonic circuits
* **Folder:** `/Project-1`

### 🔹 Project 2: Microfluidic Sensor Design

* **Description:** Development of a microfluidic-based sensor for precise detection and analysis of fluid samples.
* **Focus Areas:** Microchannel design, fluid dynamics, and sensing mechanisms
* **Applications:** Biomedical sensing, lab-on-chip systems
* **Folder:** `/Project-2`
